# Scope — kr-신화

## What this project covers

- Korean shamanic mythology (무속신화 / 서사무가), specifically 제주도 일반신본풀이
  and mainland 무가
- v0 subject: 차사본풀이 (origin of the underworld messengers) modeled deeply
  as a prototype for the rest
- Each Witness is a specific recorded transcript with named performer, recorder,
  year, and region — not a normalized "canonical version"
- Sources: T1 primary 무가 transcripts (현용준, 임석재, 김헌선, 진성기), T2
  peer-reviewed scholarship, T3 encyclopedic (한국민족문화대백과, folkency)

## What this project explicitly does NOT cover (v0)

- Other narrative families (바리공주, 천지왕본풀이, etc.) — Phase 1 expansion
- Full primary-text Korean transcripts — copyright prevents reproduction in bulk
- Full English parallel translation — deferred to v1 via Akamatsu/Akiba 1937 path
- 건국신화 (founding mythology: 단군, 동명, 혁거세) — a separate tradition
- 환단고기, 부도지, or 신민족주의 재구성 sources — excluded from canon as T5

## Source-tier discipline

T1 primary transcripts → T2 peer-reviewed → T3 encyclopedic → T4 popular →
T5 excluded. Every claim in the ontology carries a `:source` predicate pointing
to a `Source` instance with author/title/year/tier. Unsourced claims are not
admitted.
